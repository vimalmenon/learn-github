# Learn GitHub

#### Basic Commands
Check Git Version
```sh
git --version
```
Check all the files for versioning
```sh
git add .
```
Committing the message for pushing
```sh
git commit -m "<message>
```
Check Logs
```sh
git log
git log --oneline
```
